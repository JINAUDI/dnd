# Hero Layer Assets (Optional)

The live homepage now uses a single premium background image stored at `public/assets/hero/hero-bg.jpg`.  
If you later decide to return to a fully layered composition (individual sofa/armchair/chandelier PNGs), store the source layers in this directory and mirror them into `public/hero-layers/`.

Suggested filenames (if/when layers are available):

- `base-room@2x.png`
- `sofa@2x.png`
- `armchair@2x.png`
- `coffee-table@2x.png`
- `rug@2x.png`
- `chandelier@2x.png`
- `curtains@2x.png`
- `shelving@2x.png`
- `sofa-detail-1@2x.png`
- `sofa-detail-2@2x.png`

Provide @3x variants for retina displays if possible.
